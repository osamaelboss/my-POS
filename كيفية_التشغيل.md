# 🚀 كيفية تشغيل نظام نقاط البيع العربي

## الطريقة الأولى: التشغيل التلقائي (الأسهل)

### في Linux/Mac:
```bash
./run_pos.sh
```

### في Windows:
```bash
bash run_pos.sh
```

---

## الطريقة الثانية: التشغيل اليدوي

### 1. تفعيل البيئة الافتراضية:
```bash
source pos_env/bin/activate
```

### 2. تشغيل النظام:
```bash
python main.py
```

---

## 🔐 التفعيل لأول مرة

عند تشغيل النظام لأول مرة:

1. **ستظهر شاشة التفعيل**
2. **أدخل كلمة مرور المطور:** `password`
3. **اختر مدة الترخيص:**
   - 7 أيام (أسبوع)
   - 30 يوم (شهر)
   - 60 يوم (شهرين)
   - 90 يوم (3 أشهر)
   - 365 يوم (سنة)
   - أو أدخل عدد مخصص
4. **انقر "تفعيل الترخيص"**

## 📱 استخدام النظام

بعد التفعيل ستظهر الشاشة الرئيسية مع:

- **المبيعات والفواتير** - لإنشاء فواتير جديدة
- **إدارة المنتجات** - لإضافة وتعديل المنتجات
- **إدارة العملاء** - لإدارة بيانات العملاء
- **إدارة الموردين** - لإدارة الموردين
- **التقارير** - لعرض التقارير المختلفة
- **البحث السريع** - للبحث في المنتجات والعملاء
- **الإعدادات** - لإعدادات النظام

## ❗ حل المشاكل الشائعة

### المشكلة: "command not found"
**الحل:** تأكد من وجود Python 3:
```bash
python3 --version
```

### المشكلة: "No module named..."
**الحل:** تأكد من تفعيل البيئة الافتراضية:
```bash
source pos_env/bin/activate
pip install -r requirements.txt
```

### المشكلة: النظام لا يعمل
**الحل:** جرب إعادة إنشاء البيئة الافتراضية:
```bash
rm -rf pos_env
python3 -m venv pos_env
source pos_env/bin/activate
pip install -r requirements.txt
```
