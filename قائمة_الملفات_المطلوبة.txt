📁 قائمة الملفات المطلوبة لنظام نقاط البيع العربي

===============================================

🔥 الملفات الأساسية (لازم تكون موجودة):

📄 main.py                    - الملف الرئيسي للنظام
📄 requirements.txt          - قائمة المكتبات المطلوبة
📄 تشغيل_سريع.py             - ملف التشغيل السريع
📄 تشغيل.bat                 - ملف التشغيل لـ Windows
📄 run_pos.sh                - ملف التشغيل لـ Linux/Mac

📁 database/                 - مجلد قاعدة البيانات
   📄 __init__.py
   📄 excel_manager.py       - إدارة ملفات Excel

📁 ui/                       - مجلد واجهة المستخدم
   📄 __init__.py
   📄 main_window.py         - النافذة الرئيسية
   📄 license_window.py      - نافذة التفعيل

📁 utils/                    - مجلد الأدوات المساعدة
   📄 __init__.py
   📄 arabic_support.py      - دعم اللغة العربية

===============================================

📖 ملفات التوثيق (اختيارية):

📄 README.md                 - دليل المستخدم الكامل
📄 SYSTEM_SUMMARY.md         - ملخص النظام
📄 كيفية_التشغيل.md          - طريقة التشغيل بالتفصيل
📄 طريقة_التشغيل_السريعة.txt  - الطريقة السريعة

===============================================

🚫 الملفات اللي مش محتاجها:

❌ pos_env/                  - البيئة الافتراضية (هتتعمل تلقائياً)
❌ data/                     - ملفات البيانات (هتتعمل لما تشغل النظام)
❌ .git/                     - ملفات Git

===============================================

📦 طريقة التحميل:

1. حمل الملف المضغوط: arabic_pos_system.tar.gz
2. استخرج الملفات في مجلد جديد
3. افتح Terminal/Command Prompt في المجلد ده
4. شغل النظام بالأمر: python تشغيل_سريع.py

===============================================
