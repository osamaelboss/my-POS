# نظام نقاط البيع والمخزون العربي / Arabic Desktop POS System

## وصف النظام / System Description

**العربية:**
نظام نقاط البيع والمخزون المكتبي باللغة العربية مع قاعدة بيانات Excel ونظام ترخيص يدوي. يعمل النظام بدون اتصال بالإنترنت على أنظمة Windows و Linux، ويحفظ جميع البيانات في ملفات Excel فقط، ويدعم طباعة الفواتير.

**English:**
Arabic Desktop POS and Inventory Management System with Excel database and manual license control. The system works offline on Windows/Linux, saves all data in Excel files only, and supports receipt printing.

## المميزات الرئيسية / Key Features

### إدارة المنتجات / Product Management
- إضافة المنتجات (اسم عربي، كود/باركود، سعر التكلفة، سعر البيع، الكمية، تاريخ الانتهاء)
- تتبع مستويات المخزون وتنبيهات المخزون المنخفض
- تنبيهات المنتجات قاربة الانتهاء

### إدارة الموردين / Supplier Management
- تسجيل الموردين (الاسم، الهاتف، جدول التسليم، الفواتير، تواريخ الاستحقاق)
- تسجيل فواتير الموردين مع المبالغ المدفوعة والرصيد المتبقي

### إدارة العملاء / Customer Management
- تسجيل العملاء وتصنيفهم إلى مجموعات (عادي - عائلة - أصدقاء - إلخ)
- أسعار مختلفة لكل مجموعة عملاء
- إدارة المبيعات الآجلة (الشكك): اسم العميل، تاريخ الاستحقاق، الرصيد المستحق

### المبيعات والفواتير / Sales & Invoicing
- إنشاء فواتير المبيعات عن طريق اختيار المنتجات من قائمة منسدلة أو مسح الباركود
- الفاتورة تظهر: المنتجات، الكمية، سعر الوحدة، الخصم (إدخال يدوي)، الإجمالي
- طباعة الفاتورة بالعربية (حراري 80 مم أو A4)
- تسجيل المبلغ المدفوع والرصيد المتبقي

### التقارير / Reports
- الربح والخسارة اليومية (فصل تلقائي للأيام بعد 12:00 ص)
- تقارير المبيعات والمصروفات الشهرية والسنوية
- تقرير المخزون الحالي، المخزون المنخفض، المنتجات قاربة الانتهاء
- كشوف حسابات العملاء (للائتمان)

### نظام الترخيص / License System
- البرنامج يتطلب تفعيل قبل الاستخدام
- عند بدء البرنامج (إذا لم يكن مفعل): رسالة "النسخة غير مفعلة - برجاء إدخال كود التفعيل"
- كلمة مرور رئيسية (معروفة للمطور فقط)
- بعد إدخال كلمة المرور الرئيسية، شاشة لتحديد عدد الأيام (7، 30، 60 يوم)
- البرنامج يحفظ: معرف الجهاز، تاريخ البداية، تاريخ الانتهاء، الأيام المتبقية في License.xlsx
- بعد انتهاء التاريخ → البرنامج مقفل حتى إعادة التفعيل

## متطلبات النظام / System Requirements

- Python 3.6 أو أحدث / Python 3.6 or higher
- Windows 10/11 أو Linux Ubuntu 18.04+ / Windows 10/11 or Linux Ubuntu 18.04+
- 4 GB RAM (الحد الأدنى) / 4 GB RAM (minimum)
- 500 MB مساحة فارغة / 500 MB free disk space

## التثبيت / Installation

### الطريقة الآلية / Automatic Installation

1. تحميل الملفات / Download files:
```bash
git clone <repository-url>
cd arabic-pos-system
```

2. تشغيل سكريبت التثبيت / Run installation script:
```bash
python install.py
```

### التثبيت اليدوي / Manual Installation

1. تثبيت المتطلبات / Install requirements:
```bash
pip install -r requirements.txt
```

2. تشغيل النظام / Run the system:
```bash
python main.py
```

## الاستخدام / Usage

### التفعيل الأولي / Initial Activation

1. عند أول تشغيل، ستظهر شاشة التفعيل
2. أدخل كلمة مرور المطور: `password`
3. اختر مدة الترخيص (7، 30، 60، 90، 365 يوم أو عدد مخصص)
4. انقر "تفعيل الترخيص"

### الشاشة الرئيسية / Main Screen

- **المبيعات والفواتير**: إنشاء فواتير جديدة وإدارة المبيعات
- **إدارة المنتجات**: إضافة وتعديل المنتجات
- **إدارة العملاء**: إضافة وإدارة بيانات العملاء
- **إدارة الموردين**: إدارة الموردين وفواتير الشراء
- **التقارير**: عرض التقارير المختلفة
- **البحث السريع**: البحث في المنتجات والعملاء
- **الإعدادات**: إعدادات النظام

## هيكل قاعدة البيانات / Database Structure

النظام يستخدم ملفات Excel التالية في مجلد `data/`:

### Products.xlsx (المنتجات)
- كود المنتج، اسم المنتج، الباركود
- سعر التكلفة، سعر البيع، الكمية المتاحة
- الحد الأدنى للمخزون، تاريخ الانتهاء، تاريخ الإضافة

### Customers.xlsx (العملاء)
- كود العميل، اسم العميل، رقم الهاتف، العنوان
- مجموعة العميل، الرصيد المدين، تاريخ التسجيل، ملاحظات

### Suppliers.xlsx (الموردين)
- كود المورد، اسم المورد، رقم الهاتف، العنوان
- جدول التسليم، الرصيد الدائن، تاريخ التسجيل، ملاحظات

### Sales.xlsx (المبيعات)
- رقم الفاتورة، تاريخ البيع، كود العميل، اسم العميل
- كود المنتج، اسم المنتج، الكمية، سعر الوحدة
- الخصم، الإجمالي، المبلغ المدفوع، المتبقي، طريقة الدفع، ملاحظات

### Purchases.xlsx (المشتريات)
- رقم الفاتورة، تاريخ الشراء، كود المورد، اسم المورد
- كود المنتج، اسم المنتج، الكمية، سعر الوحدة
- الإجمالي، المبلغ المدفوع، المتبقي، تاريخ الاستحقاق، ملاحظات

### License.xlsx (الترخيص)
- معرف الجهاز، تاريخ البداية، تاريخ الانتهاء
- الأيام المتبقية، حالة الترخيص

## الاختصارات / Shortcuts

- `F1`: فاتورة جديدة / New Invoice
- `F2`: منتج جديد / New Product  
- `F3`: عميل جديد / New Customer
- `F4`: البحث السريع / Quick Search
- `Ctrl+P`: طباعة / Print
- `Ctrl+S`: حفظ / Save
- `Ctrl+Q`: إغلاق / Quit

## استكشاف الأخطاء / Troubleshooting

### مشاكل التثبيت / Installation Issues

**خطأ في تثبيت الحزم:**
```bash
pip install --upgrade pip
pip install -r requirements.txt --user
```

**مشكلة في الخطوط العربية:**
- تأكد من وجود خط عربي مثل Tahoma أو Arial Unicode MS
- في Linux: `sudo apt-get install fonts-dejavu fonts-liberation`

### مشاكل التشغيل / Runtime Issues

**خطأ في قاعدة البيانات:**
- تأكد من صلاحيات الكتابة في مجلد `data/`
- احذف مجلد `data/` لإعادة إنشاء قاعدة البيانات

**مشكلة في الطباعة:**
- تأكد من تثبيت برنامج طباعة PDF
- تحقق من إعدادات الطابعة

## التطوير / Development

### إضافة مميزات جديدة / Adding New Features

1. إنشاء وحدة جديدة في مجلد `ui/`
2. إضافة قاعدة بيانات جديدة في `database/excel_manager.py`
3. ربط الوحدة بالشاشة الرئيسية في `ui/main_window.py`

### بناء تطبيق تنفيذي / Building Executable

```bash
pip install pyinstaller
pyinstaller --onefile --windowed main.py
```
